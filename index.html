Computer-Administered Cognitive Tests

On this website, you will find experimental intelligence tests that I have created or co-authored.

This project began over a decade ago with the NIT-Form 1, but was discontinued around 2015.

You can find some information about NIT-Form 1 here.

It has now been revived in a new format with improved items. All tests here are self-scoring computer programs, unlike most high-range IQ tests that are scored manually.

This enables candidates to receive instant feedback, potentially making the testing experience more engaging and enjoyable for some.

All My Tests

NAIT

A 120-item, untimed, computerized test covering numerical, verbal, logical, abstract, and spatial reasoning. Quite long, but offers a very wide IQ measurement range.

NAIT-ES

A 50-item, untimed, computerized test covering the same domains as NAIT, but shorter, harder, and with an increased ceiling for higher IQ measurement.

NOACT

A 50-item, time-limited, online computerized test covering numerical, verbal, logical, and abstract reasoning with progressive difficulty.

SAT36

A 36-item, high-range spatial IQ test, co-created with Andrei Udriște.
